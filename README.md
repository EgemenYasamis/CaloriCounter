CalorieCounter

A modern calorie tracking web application that helps users monitor daily calorie intake, manage meals, and track fitness activities. The app provides a clean dashboard interface with meal planning and analytics features.

Features

Daily calorie tracking

Meal planning with breakfast, lunch, dinner, and snacks

Nutrition search integration

Fitness workout tracking

Weight and calorie analytics

Responsive dashboard interface

Tech Stack

TailwindCSS
